# shodancli
A simple go shodan command line client

[![forthebadge](https://forthebadge.com/images/badges/made-with-go.svg)](https://forthebadge.com)

# Installation
```
git clone https://github.com/krishpranav/shodancli
cd shodancli
go get
go build main.go
```

- Cross Building:
```
make all
```

# Requirements
```
go
shodan api key
```
